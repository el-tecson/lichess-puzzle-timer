## 1. Overview
Lichess Puzzle Timer is a single-purpose Chrome extension that adds a timer and simple DOM interaction to the Lichess puzzle page.  
This extension does **not** collect, transmit, store, or share any personal or sensitive user data.

---

## 2. Data Collection
**The extension does not collect any personal information.**  
It does not gather or store:

- names  
- email addresses  
- account information  
- browsing history  
- puzzle performance data  
- cookies or tracking identifiers  

No information is ever sent to external servers.

---

## 3. Local Storage
Lichess Puzzle Timer may store **local settings** (such as timer preferences) using Chromeâ€™s `storage` API.

These settings:

- remain on your device  
- are never uploaded or transmitted  
- can be cleared anytime by uninstalling the extension or clearing Chrome extension data

---

## 4. Permissions Use

### `activeTab`
Used only to run the timer script on the currently active Lichess puzzle tab.

### `storage`
Used solely to save timer settings locally on the user's device.

### Host Permissions: `https://lichess.org/*`
Required so the extension can access the Lichess puzzle page's DOM to detect puzzle events and attach the timer.  
No other sites are accessed.

---

## 5. No Remote Code
The extension does **not** fetch or execute remote code, external scripts, or remote resources.  
All functionality is packaged inside the extension.

---

## 6. Third-Party Sharing
The extension does **not** share any data with third parties.  
No analytics, advertising, tracking, or external logging is used.

## 7. Children's Privacy
Lichess Puzzle Timer does not collect personal data from any users, including children under 13.
